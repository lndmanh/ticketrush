## Parameters

::ct-field-group
  ::ct-field{name="enable" type="boolean" default-value="false"}
  Whether to enable the banner.
  ::
  ::ct-field{name="showClose" type="boolean" default-value="true"}
  Whether to show to close button.
  ::
  ::ct-field{name="to" type="string"}
  Link to navigate to. Don't set any value if you don't want to navigate.
  ::
  ::ct-field{name="content" type="string"}
  The content to show (parsed by the MDC parser).
  ::
  ::ct-field{name="target" type="Target" default-value="_blank"}
  Navigation target.
  ::
  ::ct-field{name="border" type="boolean" default-value="true"}
  Whether to show the border at the bottom of the banner.
  ::
::
