### `useConfig()`

Get the config from `docs.config.ts`.

- `main`, `header`, `aside`, `footer` and `toc` can be set from `_dir.yml` and any `page.md` file.

```ts
const {
  site,
  search,
  header,
  main,
  aside,
  footer,
  toc,
} = useConfig();
```
