---

You can override any component in the project by modifying the existing Vue component files. The project uses a component-based architecture where content components are prefixed with `Ct` and located in `app/components/content/`.

Here is a brief showcase of the `app/components/` folder structure.

::ct-file-tree
---
tree:
  - app/components:
      - layout:
            - Header.vue
            - Footer.vue
            - ...
      - content:
            - CtAccordion.vue
            - CtAlert.vue
            - CtCard.vue
            - ...
      - ui:
            - button/
            - card/
            - ...
---
::

:ct-alert[Check out the complete file structure on **GitHub**]{to="https://github.com/No-Name-Studio-VN/Homepage/tree/main/app/components" target="_blank"}

For example, if you want to override the `Footer` component, edit `app/components/layout/Footer.vue`.

```vue [app/components/layout/Footer.vue]
<template>
  <footer class="text-muted-foreground py-6 md:px-8 md:py-0">
    <div class="container flex flex-col items-center justify-between gap-2 md:h-24 md:flex-row">
      My footer
      <span class="flex-1" />
      {{ footer.credits }}
    </div>
  </footer>
</template>

<script setup lang="ts">
const { footer } = useConfig().value;
</script>
```
