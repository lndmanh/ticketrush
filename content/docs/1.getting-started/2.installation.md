## Using the starter template

::ct-steps
### Get Starter Template

:ct-pm-x{command="nuxi@latest init -t github:No-Name-Studio-VN/Nuxt-Starter-Kit"}

  ::ct-alert
  Alternatively, you can clone or download the template from the [GitHub repo](https://github.com/No-Name-Studio-VN/Nuxt-Starter-Kit).
  ::

### Navigating to the Project Directory

```bash
cd [project-name]
```

### Development Server

:ct-pm-run{script="dev"}

  ::ct-alert{type="success" icon="lucide:circle-check"}
  Well done! A browser window should automatically open for http://localhost:3000.
  ::

### Configure the docs

Pick the theme or adjust other settings to your liking.

:ct-read-more{to="/api/configuration/shadcn-docs"}

::
