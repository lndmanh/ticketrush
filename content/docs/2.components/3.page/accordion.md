## Usage

::ct-stack
  ::div{class="p-4"}
    ::ct-accordion{default-value="first-item" collapsible}
      ::ct-accordion-item{value="first-item"}
      #title
      Is it accessible?

      #content
      Yes. It adheres to the WAI-ARIA design pattern.
      ::
      ::ct-accordion-item
      #title
      Is it unstyled?

      #content
      Yes. It's unstyled by default, giving you freedom over the look and feel.
      ::
      :ct-accordion-item{title="Can it be animated?" content="Yes! You can use the transition prop to configure the animation."}
    ::
  ::
  ```mdc
  ::ct-accordion{default-value="first-item" collapsible}
    ::ct-accordion-item{value="first-item"}
    #title
    Is it accessible?

    #content
    Yes. It adheres to the WAI-ARIA design pattern.
    ::

    ::ct-accordion-item
    #title
    Is it unstyled?

    #content
    Yes. It's unstyled by default, giving you freedom over the look and feel.
    ::
    :ct-accordion-item{title="Can it be animated?" content="Yes! You can use the transition prop to configure the animation."}
  ::
  ```
::

The `value` prop in `::ct-accordion-item`{lang="mdc"} is auto-generated by default. You can set them to other _unique_ values and put them in the `default-value` prop of `::ct-accordion`{lang="mdc"}.

## Props

#### `::ct-accordion-item`{lang="mdc"}

::ct-field-group
  :ct-field{name="type" type="'single' | 'multiple'" default-value="'single'"}
  :ct-field{name="collapsible" type="boolean" default-value="undefined"}[Whether the items can be collapsed]
  :ct-field{name="defaultValue" type="string | string[]"}[Default open item]
::

#### `::ct-accordion`{lang="mdc"}

::ct-field-group
  :ct-field{name="value" type="string"}[Item value]
  :ct-field{name="title" type="string"}[Item title]
  :ct-field{name="content" type="string"}[Item content]
::
