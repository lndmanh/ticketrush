### Transformers

::ct-alert{type="info"}
**Remove the `\` in the notions to enable the transformers.**

e.g. `// [\!code --]` to `// [!code --]`
::

#### Diff

::ct-stack
::div{class="p-4"}
```ts
console.log('hewwo'); // [!code --]
console.log('hello'); // [!code ++]
console.log('goodbye');
```
::
````md
```ts
console.log('hewwo'); // [\!code --]
console.log('hello'); // [\!code ++]
console.log('goodbye');
```
````
::
